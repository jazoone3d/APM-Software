# APM

APM is a simple app that shows a live graph of Actions Per Minute (APM). You can change the graph’s color with a button.

---

## Features

- Live graph showing Actions Per Minute (APM) in real-time
- Button to change graph color
- No installation needed — just run the EXE

---

## Installation

No installation needed. Just download and open the executable file (`APM.exe`) to run the program.

---

## Usage

Run the executable to see the live APM graph.
